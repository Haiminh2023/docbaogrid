# Terms of Service – Koreader-google-drive

Plugin Koreader-google-drive chỉ giúp người dùng truy cập file trên Google Drive mà họ chọn. Plugin **không lưu trữ dữ liệu người dùng trên server nào khác**.  

Người dùng chịu trách nhiệm về dữ liệu của mình. Nhà phát triển không chịu trách nhiệm về bất kỳ mất mát hoặc lỗi nào liên quan đến dữ liệu trong Google Drive của bạn.  

Sử dụng plugin đồng nghĩa với việc bạn đồng ý với các điều khoản trên.
